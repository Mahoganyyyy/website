Occupation

Since February 2019, I have been working as a Software Developer for Mandon Software in Romford. From February 2019 to March 2020, I was enrolled on a Level 3 Software Development apprenticeship, which I successfully completed.

When I joined, I had no experience in development, except some work at school. Since I joined, I have become an integral part of a small team working on a Front-End rebuild of a powerful Insurance Broker Software used by companies spanning 3 continents.

I use the technologies and skills listed below to create front end screens that perform a wide variety of functions. Some screens are basic data represenation and entry screens, however some are more complex and have to carry out complex functions such as document production, file management, file uploading and retrieval, and any functionality our clients request.

Skills

  • C#

    I work with C# and the .NET Framework every day for our MVC based web application. I write complex procedures mainly to format large pieces of data before sending said data into the view. I can create well structured classes and models to perform a wide variety of operations.

  • JavaScript

    I use JavaScript in the views of our web app to perform any function needed on the web page. This is usually done in conjuction with jQuery and Kendo to alter the functionality of widgets, since some needed functionality is not supported out of the box by Kendo.

  • jQuery

    Although I am aware of new technologies replacing jQuery, it was essential for me to learn this technology since Kendo UI is based on it. I have learned jQuery to an intermediate level, and use it to construct Kendo UI widgets, and to collect data from forms.

  • Bootstrap 4

    I use Bootstrap whenever needed to create fully responsive forms. A large part of our web app is data entry and processing, so it is vital to have nice looking, well structured forms. I also created the navigation system for the web app using Bootstrap's navbar feature.

  • Kendo UI

    Kendo UI is a diverse front end framework that I use extensively at work. The web app has to display lots of data, so I utilise the grid widget to achieve this. I have also used other features, such as Date Pickers, Dropdowns, PDF viewers, and many more.

  • Progress OpenEdge

    Our backend is built on OpenEdge, and I have become profficient in writing REST APIs to supply data to and receive data from the web app. Most APIs are simple CRUD functions, but some are more complex and deal with larger files and images.

  • Xamarin Forms

    Recently I have been working on a Xamarin Forms application for one of our clients that serves as a dashboard portal to all of their other software. Whilst being simple, developing this app has given me a foundation level of knowledge on using Xamarin forms to develop cross-platform applications.
